# Line Following Robot using Arduino

This project is an autonomous robot that follows a predefined path using IR sensors and an Arduino microcontroller.

## Components Used
- Arduino UNO
- IR Sensors
- L298N Motor Driver
- DC Motors
- Robot Chassis
- Battery

## Working Principle
The IR sensors detect the difference between black and white surfaces. 
When the robot detects the line, the Arduino processes the sensor data and controls the motors using the L298N motor driver to keep the robot aligned with the path.

## Applications
- Autonomous robotics
- Industrial automation
- Smart transportation systems
